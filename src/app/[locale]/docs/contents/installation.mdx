# การติดตั้ง

คู่มือนี้จะแนะนำวิธีการติดตั้งโปรเจกต์บนเครื่องของคุณ

## ความต้องการของระบบ

- **Node.js**: รุ่น 18 หรือสูงกว่า
- **npm** หรือ **yarn**: สำหรับจัดการแพ็กเกจ
- พื้นที่ว่างในดิสก์: อย่างน้อย 500 MB

## ขั้นตอนการติดตั้ง

1. **โคลนรีโพสิทอรี**:
```bash
git clone https://github.com/your-repo/project.git
cd project
   ```

2. **ติดตั้ง dependencies**:
```bash
npm install
   ```

3. **รันโปรเจกต์**:
```bash
npm start
   ```

## การแก้ไขปัญหา

- **ข้อผิดพลาด "Node.js version not supported"**:
อัปเดต Node.js เป็นรุ่นล่าสุด
- **ปัญหาการเชื่อมต่อเครือข่าย**:
ตรวจสอบการเชื่อมต่ออินเทอร์เน็ตและลองรัน `npm install` อีกครั้ง
