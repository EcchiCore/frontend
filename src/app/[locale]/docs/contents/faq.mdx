# คำถามที่พบบ่อย

คู่มือนี้รวบรวมคำถามที่พบบ่อยและคำตอบเพื่อช่วยแก้ไขปัญหาของคุณ

## 1. ฉันจะเริ่มใช้งานโปรเจกต์ได้อย่างไร?

ดูรายละเอียดใน [เริ่มต้นใช้งาน](#/docs?content=getting-started)

## 2. การติดตั้งล้มเหลว ฉันควรทำอย่างไร?

- ตรวจสอบเวอร์ชันของ Node.js
- ลบโฟลเดอร์ `node_modules` และรัน `npm install` อีกครั้ง
- ดูรายละเอียดเพิ่มเติมใน [การติดตั้ง](#/docs?content=installation)

## 3. ฉันสามารถปรับแต่งธีมได้หรือไม่?

ได้! คุณสามารถปรับแต่งธีมได้ในไฟล์ `config.json` ดูวิธีการใน [ฟีเจอร์ขั้นสูง](#/docs?content=advanced-features)

## 4. มีการสนับสนุนสำหรับผู้ใช้หรือไม่?

ติดต่อทีมสนับสนุนผ่านอีเมล: support@example.com
