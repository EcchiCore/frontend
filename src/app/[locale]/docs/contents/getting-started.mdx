# เริ่มต้นใช้งาน

ยินดีต้อนรับสู่คู่มือเริ่มต้นใช้งานโปรเจกต์ของเรา! คู่มือนี้จะช่วยให้คุณเข้าใจพื้นฐานและเริ่มใช้งานได้อย่างรวดเร็ว

## ภาพรวม

โปรเจกต์นี้เป็นเครื่องมือที่ออกแบบมาเพื่อช่วยให้คุณทำงานได้อย่างมีประสิทธิภาพมากขึ้น โดยมีฟีเจอร์หลักดังนี้:

- **ใช้งานง่าย**: อินเทอร์เฟซที่เป็นมิตรต่อผู้ใช้
- **ปรับแต่งได้**: รองรับการปรับแต่งตามความต้องการ
- **ประสิทธิภาพสูง**: ทำงานได้รวดเร็วและเสถียร

## ขั้นตอนแรก

1. **ตรวจสอบความต้องการของระบบ**:
- Node.js รุ่น 18 หรือสูงกว่า
- ระบบปฏิบัติการ: Windows, macOS, หรือ Linux
2. **ติดตั้งโปรเจกต์**:
ดูรายละเอียดใน [การติดตั้ง](#/docs?content=installation)
3. **ทดลองใช้งาน**:
รันคำสั่ง `npm start` เพื่อเริ่มต้น

## ตัวอย่างโค้ด

```jsx
function HelloWorld() {
  return <h1>สวัสดี, โลก!</h1>;
}
```
